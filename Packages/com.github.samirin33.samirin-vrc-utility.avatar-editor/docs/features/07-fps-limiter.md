# FPS Limiter

## 概要

**FPS Limiter** は、オブジェクトにアタッチせずに使えるエディタ拡張です。独立したエディタウィンドウから目標 FPS をスライダーやプリセットで設定でき、Unity エディタ上での最大 FPS を制限できます。設定は EditorPrefs に保存されるため、ドメインリロード後も保持されます。

## 開き方

- メニュー **`samirin33 Editor Tools`** → **`FPS Limiter`** を実行し、FPS Limiter ウィンドウを表示

## 使い方

1. **samirin33 Editor Tools** → **FPS Limiter** でウィンドウを開く
2. **目標 FPS** をスライダー（1〜120）で指定するか、プリセットボタンで設定
3. 変更は即座に反映され、`Application.targetFrameRate` が更新されます

### 設定項目

- **目標 FPS**: スライダーで 1〜120 の範囲で指定
- **プリセット**: **10 FPS** / **30 FPS** / **60 FPS** / **120 FPS** / **無制限** の各ボタンで一括設定（無制限は内部で -1）

### 永続化

- 設定値は EditorPrefs に保存されるため、Unity の再起動やドメインリロード後も前回の目標 FPS が維持されます。

## 注意事項

- この機能は**エディタプレビュー専用**です。ビルドには影響しません。ギミックの動作確認にのみ活用できます。
